# python -- 计算 平方，乘方，平方根


**1.计算乘方**

```python
pow(4,3)
# 结果64
```

**2.计算平方**

```python
import numpy
numpy.square(4)
# 结果16
```

```python
pow(5,2)
#结果25
```

**3.平方根**

```python
import numpy
numpy.sqrt(16)
# 结果4.0
numpy.sqrt(16.)
# 结果4.0
```

```python
pow(25, 0.5)
#结果5.0
pow(25, .5)
#结果5.0
```

```python
import math
math.sqrt(25)
#结果5.0
math.sqrt(25.0)
#结果5.0
```
